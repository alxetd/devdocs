# Remote

## Commands

### git remote
```bash
git remote
git remote show origin # If referential integrity is intact
git remote set-url origin git@bitbucket.org:USERNAME/REPOSITORY.git
git remote set-url origin git@bitbucket.org:USERNAME/REPOSITORY.git --push
git remote rename origin upstream	 
git remote add origin git@bitbucket.org:USERNAME/REPOSITORY.git	
git remote add upstream git@bitbucket.org:USERNAME/REPOSITORY.git	
git remote -v
```
